# Thunder-Chat-Server

## How to start?

### Development

```
git clone https://github.com/JIHYO-JEON/Thunder-Chat-Server.git
cd Thunder-Chat-Server
npm i
npm run dev
```

### Firebase emulator

```
npx firebase emulators:start
```
